# Spin Again

A solo, fake-money casino simulator with a black-and-white doodle style.

## Included games

- Slots
- Blackjack
- European roulette
- Coin flip
- Five-horse race

## Features

- No accounts, backend, paid services, or dependencies
- Browser-based fake balance and statistics saved with `localStorage`
- Random outcomes generated with `crypto.getRandomValues()`
- Responsive layout for desktop and mobile
- Hand-drawn SVG artwork based on the generated doodle models
- GitHub Pages deployment workflow

## Run locally

Open `index.html`, or serve the folder with any static web server.

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Important

This project is a simulation. It does not support real money, purchases, prizes, deposits, or withdrawals.

The original visual models were generated for this project with OpenAI image generation, then recreated as lightweight SVG game assets.
